# Lists vs Dictionaries

## Lists
A list is used to store multiple values in a specific order. Each item in a list has a position called an index, starting from 0. You access items using their index number. Lists are useful when the order of items matters or when you just want to store a collection of values.

## Dictionaries
A dictionary stores data as key–value pairs. Instead of using positions like a list, you use a key to access the value. Dictionaries are useful when each piece of data has a label or meaning attached to it.

## When to Use Each

### Use a list when:
- You have a collection of items
- Order matters
- Items are similar (like scores, numbers, or names)

### Use a dictionary when:
- You want to describe something with attributes
- Each value has a label
- You need to quickly access data using a key
